# Netflix Userbase Analysis Project
## 📌 Project Overview
This repository contains a comprehensive analysis of a simulated Netflix user database. The project explores subscriber demographics, revenue trends, and usage patterns across different countries, age groups, and devices between 2020 and 2024.

The goal of this assignment is to perform data exploration and visualization to understand which factors contribute most to Netflix's revenue and user growth.

## 📊 Key Insights at a Glance
Total Subscribers: 2,500

Total Revenue: $31,271

Primary Subscription Types: Basic, Standard, and Premium.

Device Usage: Analysis across Laptops, Smart TVs, Smartphones, and Tablets.

Demographics: Even distribution between Male (1,243) and Female (1,257) subscribers.

## 📂 Dataset Description
The analysis is based on the Netflix Userbase.csv dataset, which includes the following attributes:

User ID: Unique identifier for each subscriber.

Subscription Type: The plan chosen by the user (Basic, Standard, Premium).

Monthly Revenue: The amount paid by the user per month.

Country: The geographic location of the user (10 countries represented).

Age: User age, ranging from 26 to 51 years.

Gender: Male or Female.

Device: The primary device used for streaming.

Payment Date: Historical record of subscription payments.

## 🗂️ Project Structure
The repository is organized as follows:

/data: Contains the raw CSV files and yearly breakdowns (2020-2024).

Netflix Userbase.csv: The primary dataset.

Pivot_Analysis.csv: Summarized data including revenue by device and age group.

Yearly_Reports/: Specific data subsets for years 2020, 2021, 2022, 2023, and 2024.

## 📈 Analysis Goals
Revenue Trends: Identify which subscription plan generates the most revenue (Currently: Basic at $12,469).

Geographic Distribution: Compare user counts and revenue across countries like the USA, Canada, UK, and Australia.

Device Preference: Determine the most popular streaming devices (Laptop and Tablet are leading categories).

Age Demographics: Analyze how different age brackets (26-30, 31-35, etc.) engage with the service.

## 🛠️ Technologies Used

Python (Pandas/Matplotlib): For advanced data cleaning and visualization.

Markdown: For project documentation.
